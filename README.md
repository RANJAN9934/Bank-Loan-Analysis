# Bank-Loan-Analysis
# 📊 Financial Domain Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Financial Analysis Dashboard** developed using **Microsoft Power BI**. The dashboard provides comprehensive insights into financial performance by analyzing key business metrics such as revenue, profit, sales trends, customer behavior, and product performance.

The objective of this project is to transform raw financial data into meaningful business insights that support strategic decision-making.

---

## 🚀 Project Objectives

- Analyze overall financial performance.
- Monitor revenue and profit trends.
- Identify top-performing products and customers.
- Track sales performance across different dimensions.
- Build an interactive dashboard for business users.
- Enable quick decision-making using KPIs and visual analytics.

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query (Data Cleaning & Transformation)
